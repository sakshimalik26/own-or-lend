# Own or Lend?

An interactive classroom simulator that introduces shares and bonds to first-time finance learners through the story of a fictional Campus Café.

## What students can explore

- Share price, investment, ownership percentage, and voting power
- The difference between owning a company and owning its individual assets
- Bond principal, annual interest, maturity, and repayment risk
- Assets, liabilities, owners' equity, and limited liability
- Profit, loss, growth, and business-closure scenarios
- A quick shares-versus-bonds knowledge check

## Run locally

Requires Node.js 22.13 or newer.

```bash
npm install
npm run dev
```

Then open the local address shown in the terminal.

## Build

```bash
npm run build
```

## Educational note

This project is intended for classroom learning and does not provide investment advice. The examples simplify some legal and accounting details so beginners can understand the core concepts.
